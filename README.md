# UAS Pemograman Web 2
## Perpustakaan Digital (Hogwarts-library)

Sistem menyediakan beberapa fitur utama, yaitu pengelolaan data kategori, data buku, data anggota, transaksi peminjaman, pengembalian buku, serta pengelolaan denda. Seluruh fitur tersebut hanya dapat diakses oleh admin setelah berhasil melakukan proses autentikasi melalui halaman login.
